## 2. Updating Content


**Adding new content:**
1. Open the relevant HTML file
2. Copy the HTML structure of an existing card
3. Add  relevant image to images folder
4. Update image src, name, details
5. Test that layout still works
6. Commit and push changes


**Updating existing content:**

1. Open the relevant HTML file
2. Find the section to update using Find feature
3. Change the text/information
4. If updating image, replace file in images folder with same name
5. Test page displays correctly
6. Commit and push changes


**Removing outdated content:**

1. Delete relevant card section from HTML file/files 
3. Remove image from images folder (or keep for archive)
6. Commit and push changes
